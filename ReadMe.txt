For this project to work 
1-)you need to include 2 jars inside extLibraries folder to java build path
2-)You need to have an sql server installed with username:root password:root Can be changed in SimpleDatabaseManager
3-)You need to run the main of SimpleDatabase.java 
*if nothing is wrong it will create a database named "PublicKeys" and a table named "Certificate"
4-)You need to run PKIServer.java
5-)You need to run Client.java
*For all commands to work correctly you need to obtain-server-publickey and (create-keypair or read-keypair)
